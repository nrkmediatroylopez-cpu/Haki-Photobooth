📧 Email Configuration

Configure EmailJS to send strips automatically. Get a free account at emailjs.com

📋 EmailJS Template Variables
Use these in your EmailJS template:
{{to_email}} — recipient email
{{from_name}} — booth name
{{subject}} — email subject
{{message}} — body text
{{strip_image}} — base64 strip image

⚠️ Note: Free EmailJS sends attachments as links. For file attachments, upgrade to a paid plan or use a backend server (see README).
